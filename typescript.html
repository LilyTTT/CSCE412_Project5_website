What is TypeScript and Why is it Used?

TypeScript(TS) is a statically typed superset of JavaScript(JS) that adds optional static typing and other features to the JavaScript language. TypeScript compiles to plain JS code, which means it can run in any JS environment, making it highly compatible with existing web and Node.js applications.
